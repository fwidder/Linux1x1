# 007:

Die Bank:

Schreibe eine Skript / File Basierte Kontoverwaltung für eine Bank:

Es soll möglich sein:
 - Konto erstellen
 - Konto Schliessen
 - Ein- und Auszahlung
 - Überweisungen
 - Konto Informationen Anzeigen

Zu jedem Konto sollen Kontonummer, Inhaber sowie der aktuelle Saldo gespeichert werden (Keine Historie)
Um das ganze zu vereinfachen sollen nur ganze Euro beträge verwaltet werden.
Zudem wird davon asugegangen das der Nutzer (abgesehen von falschen Kontonummern) keine Fehleingaben macht.
